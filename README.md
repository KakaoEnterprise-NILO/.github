# Gridam
## 서비스 소개
<img width="1385" height="819" alt="image 220" src="https://github.com/user-attachments/assets/e0397c12-319b-467d-8ddc-1ab11ec67896" />

#### "하루의 기억들을 그리듯이 저장해요. 모인 기억들은 추억이 될 수 있게"
감정을 기록하고 공유하는 감상 아카이브 플랫폼

## 주요 기능
- 개인 일기 저장 및 확인
- 일기 내용 기반으로 한 감정 카드 생성
- 피드 공유
- 캘린더 및 감정 분석 통계
- 알림
